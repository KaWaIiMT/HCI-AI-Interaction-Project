# UI改造计划

> 目标：在保留底层功能的基础上，让界面"看起来"完全是一个新项目
>
> 原则：轻量改皮，不改动 Vue 组件架构

---

## 一、配色方案

### 原项目配色
- 主色调：粉紫色系（#ff6b9d 等）
- 风格：二次元/可爱风

### 新配色方案（科技蓝）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#1a73e8` | Google Blue，科技感 |
| 主色深 | `#1557b0` | Hover/Active 状态 |
| 辅助色 | `#5f6368` | 文字/图标 |
| 背景色 | `#f8f9fa` | 浅灰白背景 |
| 侧边栏背景 | `#1e293b` | 深色侧边栏 |
| 卡片背景 | `#ffffff` | 白色卡片 |
| 成功/确认 | `#0f9d58` | 绿色 |
| 警告 | `#f4b400` | 黄色 |
| 错误 | `#db4437` | 红色 |

### CSS 修改文件
- `static/css/styles.css` — 主样式文件
- `static/css/vrm.css` — VRM 相关样式
- `static/css/transition.css` — 过渡动画

---

## 二、Logo/图标

### 原项目Logo
- 猫/二次元角色头像（位于 `static/source/` 目录）

### 新Logo方案
- 设计一个简洁的几何图标（可参考「A」+「D」字母组合）
- 或使用 AI 生成一个科技感 icon
- 替换文件：
  - `static/source/icon.png` — 主图标
  - `static/source/icon.icns` — macOS 图标
  - `static/source/icon_tray.png` — 托盘图标
  - `static/source/coveren.png` — README 封面（删除/替换）

---

## 三、页面标题与文案

### 需修改的位置（`static/index.html` 中搜索）
- `<title>` 标签内容
- 应用主标题/副标题
- 关于对话框
- 版本号显示

## 四、侧边栏菜单改造

### 原菜单结构（推测）
```
🏠 首页/对话
🐱 桌面宠物
🗨️ 群聊/酒馆 ← 删除入口
📋 任务中心
🔌 即时通讯 ← 删除入口
📺 直播 ← 删除入口
🎮 扩展商店 ← 删除入口
⚙️ 设置
```

### 新菜单结构（建议）
```
🏠 智能对话
🐱 桌面伙伴
📋 任务管理
📚 知识库
⚙️ 系统设置
```

### 修改位置
- `static/index.html` 中的侧边栏模板
- `static/js/vue_data.js` 中的菜单数据
- `static/js/vue_methods.js` 中的路由/菜单逻辑

---

## 五、默认页面

- 启动后默认展示 **智能对话** 页面
- 桌面宠物常驻显示（如果有开关）
- 首次启动显示欢迎引导（可选添加）

---

## 六、不需要改的部分

- VRM 渲染管线（`static/vrm.html`、`static/js/vrm.js`）
- 聊天消息渲染逻辑
- WebSocket 通信层
- 设置保存/读取逻辑
- Markdown 渲染组件
- 所有 `static/libs/` 下的第三方库

---

## 七、执行优先级

1. **高优先级**（必须改，否则一眼看穿）
   - 配色方案
   - Logo/图标
   - 页面标题
   - 侧边栏菜单

2. **中优先级**（建议改）
   - 关于对话框
   - 版权信息
   - 默认页面布局微调

3. **低优先级**（可选）
   - 动画效果调整
   - 字体替换
   - 启动画面
